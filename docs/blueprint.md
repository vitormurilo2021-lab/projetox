# **App Name**: Casei Barato

## Core Features:

- Dynamic Landing Page Display: Render all required landing page sections (Hero, Problem, Solution, How it Works, Offers, Transformation, Guarantee, Closing CTA) with a mobile-first, responsive design approach.
- Interactive Wedding Cost Quiz: Engage users with a multi-step quiz to collect data necessary for the wedding cost simulation.
- Intelligent Wedding Cost Simulator: Process quiz inputs to provide users with a detailed estimated wedding budget and expenditure breakdown, demonstrating cost categories.
- Personalized Cost-Saving Strategy Tool: An AI-powered tool that generates tailored recommendations for where to cut costs and how to prioritize spending based on the user's simulation results, aligning with the Premium plan's value propositions like 'decisões prontas' and 'estratégias para economizar mais'.
- Subscription Plan Comparison & Selection: Display the 'Plano Essencial' and 'Plano Completo' cards, visually highlighting the 'Plano Completo' with appropriate badges and feature lists, allowing users to choose their preferred subscription.
- Secure Payment & Subscription Integration: Facilitate secure handling of user subscriptions for chosen plans and process payments through a third-party gateway.
- Conditional Upgrade Pop-up: Present a strategic upgrade offer pop-up to users attempting to subscribe to the 'Plano Essencial', detailing the value of upgrading to the 'Plano Completo' at a discounted price.

## Style Guidelines:

- The overall scheme will be light to emphasize clarity, trustworthiness, and conversion. Primary Color: A deep, authoritative blue (#244480) for its professional and intelligent associations, providing strong contrast for calls-to-action on a light background. Background Color: A very subtle, almost white blue-gray (#F2F6FA) for a clean and open feel, ensuring content readability. Accent Color: A vibrant, elegant purple (#9E40FF) for elements requiring strong emphasis or a touch of premium feel, offering compelling contrast with the primary blue.
- Headline font: 'Playfair' (serif) for an elegant, fashionable, and aspirational feel, suitable for capturing attention in key titles. Body text font: 'PT Sans' (sans-serif) for its modern readability, warmth, and practicality, ensuring clarity in longer content blocks and mobile displays.
- Use modern, clean line icons. Prominently feature checkmark icons for plan benefits. Incorporate visually engaging badges such as 'MAIS ESCOLHIDO' (Most Chosen) on the Premium plan card.
- Strictly mobile-first responsive design. Key sections like problem, solution, and transformation should flow vertically. The offer cards should be presented side-by-side on desktop for comparison, but elegantly stacked on mobile devices. Ensure ample whitespace to maintain a clean and uncluttered user interface. The upgrade pop-up will be a prominent, centered overlay with clear primary and secondary action buttons.
- Implement subtle, unobtrusive animations such as smooth scrolling to sections and gentle fades or slides for content reveal to enhance user engagement without distraction. The upgrade pop-up will feature a smooth entrance animation for a compelling user experience.
